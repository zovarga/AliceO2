InjectorFunction o2::utilities::dataSamplingReadoutAdapter ( framework::OutputSpec const &  spec)

An adapter function for data sent by Readout for Data Sampling / Quality control purposes. It adds DataHeader and DataProcessingHeader using information found in received DataBlockHeaderBase header and given OutputSpec. It IGNORES SubSpecification in OutputSpec and uses linkID instead.
