230 // Helper class to requested data.
231 // Most common data have dedicated getters, some need to be called with returned typa as a template.
232 // In general on either gets a gsl::span<const Type> via getter like e.g. getITSTracks()
233 // of a reference to particular track, i.e. getITSTrack(GlobalTrackID id).
234 // Note that random access like getITSTracks()[i] has an overhead, since for every call a span is created.
235 // Therefore, for the random access better to use direct getter, i.e. auto& tr = getITSTrack(gid)
236 // while for looping over the whole span first create a span then iterate over it.
