Detailed Description

A more familiar task API for the DPL. This allows you to define your own tasks as subclasses of o2::framework::Task and to pass them in the specification using:

adaptTask<YourDerivedTask>(constructor args, ...);

The appropriate AlgorithmSpec invoking Task::init(...) at startup and Task::run(...) will be created.
