72  This is is done with a configuration string like the following
73  one, where the input matching for each device is provide in

75  A:TRD/RAWDATA/785;B:TRF/RAWDATA/2560,C:TRD/RAWDATA/1280;D:TRD/RAWDATA/1536

77  will lead to a workflow with 2 devices which will input match
78 
79  trd-crucompressor-0 --> A:TRD/RAWDATA/768;B:TRD/RAWDATA/1024
80  trd-crucompressor-1 --> C:TRD/RAWDATA/1280;D:TRD/RAWDATA/1536
81  The number after the RAWDATA is the FeeID in decimal
82  */

86  int idevice = 0;

88  LOG(info) << "for now ignoring the multiple processors, something going wrong";
