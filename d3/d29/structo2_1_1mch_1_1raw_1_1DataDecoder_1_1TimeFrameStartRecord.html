Detailed Description

Structure holding the value of the BC counter from the last decoded Heartbeat packet from a given SAMPA chip, as well as the value of the first orbit in the corresponding TimeFrame The structure also keeps track of the previous orbit/bc pair, in order to perform consistency checks
